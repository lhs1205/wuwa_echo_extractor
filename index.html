게임 전체화면 스크린샷에서 에코 부옵션을 무료로 자동 추출합니다 · 최대 5개 동시 처리

사용 방법
1 게임에서 에코 인벤토리를 열고 부옵션을 확인할 에코를 선택합니다.
에코 상세 패널이 화면 우측에 표시된 상태여야 합니다.
2 전체화면 스크린샷을 찍습니다.
Windows: Win + PrtSc 또는 PrtSc · 16:9 해상도 권장 (1920×1080 등)
3 아래 업로드 영역에 스크린샷을 드래그&드롭하거나 클릭해서 파일을 선택합니다.
한 번에 최대 5개까지 업로드해 동시에 처리할 수 있습니다.
4 분석 시작 버튼을 누릅니다.
첫 실행 시 한국어 OCR 데이터(~5MB)를 다운로드하므로 잠시 기다려 주세요.
5 추출된 부옵션을 확인하고 📋 복사 버튼으로 클립보드에 복사합니다.
스탯명·수치를 클릭하면 직접 수정할 수 있습니다. 인식이 어긋난 경우 수정 후 복사하세요.
